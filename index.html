how to host your domain on github in 5 steps?
  • 1. in your domain provider, have the domain pointing to github by creating :
    • 4 times A Record @ 185.199.108/9/10/11.153
  • 2. in your github account repositories tab,
    • create a new repository and give it name (now you could now just clone it)
  • 3. in its code tab,
    • have the cname file containing your new site name
  • 4. in its settings tab, in its GitHub Pages,
    • chose Source to be master branch, Custom domain your domain and tick enforse https
  • 5. wait some time and then in its code tab 
    • rewrite your site by uploading your new index.html file (edited with the composer of seamonkey -editor ).
